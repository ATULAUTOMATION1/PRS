# PRS GROUP Website

## 🚀 Deployment Instructions (Hostinger via GitHub)

This website is a **Next.js** project optimized for **Static Export**.

### 1. Push to GitHub
If you haven't already, push this codebase to your GitHub repository:
```bash
git init
git add .
git commit -m "feat: complete prs group conglomerate branding"
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```

### 2. Configure Hostinger
When setting up the deployment on Hostinger:
*   **Repository:** Select your GitHub repository.
*   **Build Command:** `npm run build`
*   **Output Directory:** `out` (This is critical for Next.js static export).
*   **Install Command:** `npm install`
